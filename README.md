# StudentInfo
> 基于SSM的学生信息管理系统(选课 + 评教)

## 运行环境
操作系统：Linux、Mac os、windows
支持环境：JDK 1.8、Maven3
数 据 库：Mysql

## 使用技术
* 工具：IDEA、sequel Pro
* 环境：JDK1.8、tomcat9.0、mysql8.0
* 前端：JavaScript、jQuery、bootstrap4、particles.js
* 后端：maven、SpringMVC、MyBatis、ajax、mysql读写分离、mybatis分页
